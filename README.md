# 以 rlutil.h 實作文字版 UI 成績計算器

## 摘要
本期末專題使用 C 語言搭配 rlutil.h 函式庫，實作一個文字版使用者介面的成績計算器。透過游標定位、文字顏色控制與鍵盤輸入操作，提升傳統主控台程式的互動性與可讀性，並加深對主控台 UI 設計與程式結構的理解。

## 過程簡述
本專題首先規劃程式功能，決定以選單式介面作為主要操作方式。接著學習 rlutil.h 提供的相關函式，如清除畫面、設定文字顏色、移動游標位置與讀取鍵盤輸入等，建立基本的文字 UI 架構。完成介面後，再逐步加入成績輸入、資料統計與錯誤處理等功能，並透過反覆測試修正程式行為。

## 內容與結果
本程式提供以下功能：
- 新增成績（輸入範圍為 0 至 100）
- 列出所有已輸入的成績
- 計算成績統計資料（平均、最大值、最小值、及格率）
- 清除所有成績資料

使用者可透過鍵盤方向鍵選擇功能，並以 Enter 鍵確認操作。畫面中使用不同顏色區分標題、選單與提示訊息，使介面資訊層次更加清楚。實作結果顯示，透過 rlutil.h 能有效改善文字介面的操作體驗，使主控台程式更具互動性。
